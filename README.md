# PartyScore Karaoke

Instant karaoke party game: sing into your mic → get a fun score → share & challenge friends.

**Live demo (GitHub Pages):** https://manulus1-ai.github.io/clawd-build-20260131-2128Z-partyscore-karaoke/

## Stack
- Frontend: Angular (static)
- Backend: none for this MVP (frontend-only)

## Local dev

```bash
cd frontend
npm install
npm start
```

## Build for GitHub Pages

```bash
cd frontend
npm run build -- --configuration production --base-href "/clawd-build-20260131-2128Z-partyscore-karaoke/"
```

## Notes
- Mic access required for gameplay.
